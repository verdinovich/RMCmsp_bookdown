Basin Average Precipitation-Frequency Using Max Stable Processes

B Example Workflow

B.1 Using the Data from RMCmsp

This appendix presents an example workflow using the data that comes baked into the R package. To begin, double click the RStudio Project File in the RMCmsp folder: RMCmsp.Rproj. This opens up an instance of RStudio. You should see “RMCmsp - RStudio” in the top left banner of RStudio. You will need to have devtools installed. From RStudio, type install.packages("devtools") into the Console. Once that is successfully installed, you should be able to run the following workflow:

library(devtools)
load_all()

# Step 1
msp_workspace   <- msp_setup(anmax=mc_anmax, metadata=mc_meta, prism_rasters=mc_prism, subbasin_targets=mc_targets)
plot_at_site_gev(at_site_gev = msp_workspace$gev.fits, shp=mc_shapefile)

# Step 2
msp_fit         <- fit_msp(msp_workspace=msp_workspace, covar.mod='twhitmat')

# Step 3
msp_uncert_mvn  <- msp_uncertainty(msp_fit=msp_fit, method="mvn", nsims=100)

# Step 4
msp_sims_mvn    <- simulate_from_msp(msp_uncertainty=msp_uncert_mvn, msp_fit=msp_fit, nsims=10000, subbasin_targets=mc_targets, target_weights=mc_weights)

# Bonus Functions
msp_smoothed    <- msp_smooth(msp_sims=msp_sims_mvn)
msp_pt_precip   <- msp_point_precip(msp_fit=msp_fit, shp=mc_shapefile, mapping_aep=0.01, tabular_aeps=NULL)

Easy right? :)

B.2 Using Your Own Data from Section 2

This appendix presents an example workflow using the data you put together in Section 2. To begin, double click the RStudio Project File in the RMCmsp folder: RMCmsp.Rproj. This opens up an instance of RStudio. You should see “RMCmsp - RStudio” in the top left banner of RStudio. You will need to have devtools installed. From RStudio, type install.packages("devtools") into the Console. Once that is successfully installed, you should be able to run the following workflow:

library(devtools)
load_all()

# Load the data
anmax    <- read.csv(paste0(idir, "atsite-anmax-",dur,"-day.csv"))
metad    <- read.csv(paste0(idir, "atsite-anmax-",dur,"-day-metadata.csv"))
prism    <- readRDS(paste0(idir,  "prism_crop.rds"))
targets0 <- read.csv(paste0(idir, "target_locations.csv"))
weights  <- targets0$area
targets  <- subset(targets0, select=c("lon","lat"))

# shapefile is optional
shp      <- terra::vect(paste0(idir, "basin.shp"))

# Step 1
msp_workspace   <- msp_setup(anmax=anmax, metadata=metad, prism_rasters=prism, subbasin_targets=targets)
plot_at_site_gev(at_site_gev = msp_workspace$gev.fits, shp=shp) # if you don't have shp, just remove that argument

# Step 2
msp_fit         <- fit_msp(msp_workspace=msp_workspace, covar.mod='twhitmat')

# Step 3
msp_uncert_mvn  <- msp_uncertainty(msp_fit=msp_fit, method="mvn", nsims=100)

# Step 4
msp_sims_mvn    <- simulate_from_msp(msp_uncertainty=msp_uncert_mvn, msp_fit=msp_fit, nsims=10000, subbasin_targets=targets, target_weights=weights)

# Bonus Functions
msp_smoothed    <- msp_smooth(msp_sims=msp_sims_mvn)
msp_pt_precip   <- msp_point_precip(msp_fit=msp_fit, shp=shp, mapping_aep=0.01, tabular_aeps=NULL)
