Basin Average Precipitation-Frequency Using Max Stable Processes

Chapter 2 Precipitation Database

The example workflow presented in this document relies on daily precipitation observations from the Global Historical Climatology Network (GHCN; Menne et al. 2012). Download the station inventory for the GHCN-daily dataset (GHCNd). Pull the data into a spreadsheet and save as a csv named ghcnd-inventory.csv. Give the columns the following names: ghcnd_id, lat, lon, varid, startyr, endyr. Note that the startyr and endyr columns aren’t used in this analysis. Note that you need to define two sub-directories within the MSP analysis directory named Data and input_files.

df <- read.csv("ghcnd-inventory.csv")

2.1 Download GHCNd stations

Plot the stations in GIS with your basin shapefile and pertinent raster data to assist in the definition of a study region surrounding the basin. Consult a hydrometeorologist (such as John England, Samantha Hartke, or Andrew Verdin) if you need help defining a bounding box. Once you have the bounding box defined, create a subset of the station inventory and download the data for those stations using the following R code (0a_download_ghcnd.R):

# set your working directory to the folder you want to save all the GHCNd data
setwd(“C:/Path/To/MSP/Analysis/Data/”)

# define the output directory - this will be the input to the MSP, call it "input_data"
idir = "C:/Path/To/MSP/Analysis/input_data/"

# read in the inventory file, do NOT put the inventory file in your data folder. 
df.m = read.csv(“C:/Path/To/MSP/Analysis/ghcnd-inventory.csv”)  

# take subset of inventory file; constrain to bounding box and only PRCP stations
# bounding box: -85 to -82.8 longitude; 33.5 to 35.5 latitude
df.sub = subset(df.m, lon > -85 & lon < -82.8 & lat > 33.5 & lat < 35.5)
df.sub = subset(df.sub, varid == "PRCP")

# set up a progress bar to gauge how long it will take
pb = txtProgressBar(min=0,max=nrow(df.sub),style=3)

# loop through each row of the subset and attempts to download a file for that station.
# destfile is saving it to your current working directory, use getwd() to check where
for(i in 1:nrow(df.sub)){
  setTxtProgressBar(pb,i)
  download.file(url=paste0("https://www.ncei.noaa.gov/data/global-historical-climatology-network-daily/access/",df.sub$ghcnd_id[i],".csv"),
                destfile=paste0(df.sub$ghcnd_id[i],".csv"))
}

# save the subset file for later use. do NOT save the subset file to the data folder
write.csv(df.sub, paste0(idir, "GHCNd-metadata.csv"), row.names=FALSE)

2.2 Combine and Organize GHCNd Data

After downloading all the data for the stations in the bounding box, that data needs to be extracted and put into a (more) concise format. The first step is to loop through all the files to define the earliest and latest date to set up a matrix that will hold all sites (rows) and observations (columns) for every day from the earliest to the latest date. This will be a huge matrix! Use the following R code for this task (0b_ghcnd_dataset.R):

library(lubridate)
library(plyr)

# read in that subset we made from the station inventory and bounding box
df.meta = read.csv(paste0(idir,"GHCNd-metadata.csv"))

# fn is a list of all filenames; fns is the same with “.csv” removed (just site ID)
fn = list.files(pattern=”.csv”)
fns = unlist(strsplit(fn, split=".csv"))

# start with some arbitrary min and max dates – update as needed for your analysis
mind = ymd(“2005-01-01”)
maxd = ymd(“2005-12-31”)

# loop through and update ‘mind’ and ‘maxd’ variables as needed
for(i in 1:length(fn)){
  df = read.csv(fn[i])
  if(min(ymd(df$DATE)) < mind) mind = min(ymd(df$DATE))
  if(max(ymd(df$DATE)) > maxd) maxd = max(ymd(df$DATE))
}

# define vector of all dates, and the matrix that will hold all the data
full_dates = data.frame(DATE=seq.Date(from=ymd(mind), to=ymd(maxd), by=”day”))
df.daily = matrix(NA, nrow=length(fn), ncol=nrow(full_dates))

# loop through and populate that matrix
pb = txtProgressBar(min=0,max=nrow(df.daily),style=3)
for(i in 1:nrow(df.daily)){
  setTxtProgressBar(pb,i)
  tmp = read.csv(fn[i])
  sub = subset(tmp, select=c("DATE","PRCP"))
  sub$DATE = ymd(sub$DATE)
  tmp.full = join(full_dates, sub, by="DATE")
  df.daily[i,] = tmp.full$PRCP
}

# save it. do NOT save the data matrix to the data folder
write.csv(df.daily, paste0(idir,“ghcn-daily.csv”), row.names=FALSE)

2.3 Calculate Annual Maxima

Now that the daily precipitation observations have been put into the big data matrix, it’s time to process those annual maxima. For this approach we are looking at the annual maximum n-day precipitation observation at each site independently. Use the following R code to create the n-day annual maxima (0c_calc_anmax.R):

library(zoo)
library(plyr)

# define output matrix
anmax.out = matrix(NA, nrow=nrow(df.daily), ncol=length(unique(year(ymd(full_dates$DATE)))))

# 2-day critical duration (currently only works for daily scale)
dur = 2  

# loop through each row of df.daily and calculate annual maxima
pb = txtProgressBar(min=0, max=nrow(df.daily), style=3)
for(i in 1:nrow(df.daily)){
  setTxtProgressBar(pb,i)

  tmp.stn = data.frame(DATE=full_dates$DATE, prcp=as.vector(t(df.daily[i,])))
  tmp.stn$yr = year(ymd(tmp.stn$DATE))
  tmp.stn$nday = rollsum(tmp.stn$prcp, k=dur, align="right", na.pad=TRUE)
  tmp.out = ddply(tmp.stn, .(yr), summarise, maxp = max(nday,na.rm=TRUE))
  tmp.out$maxp[tmp.out$maxp < 0] = NA
  anmax.out[i,] = as.vector(t(tmp.out$maxp))
}

You should only save those sites that have at least ~20 years of data, as follows:

# boolean indicating which sites have at least 20 observations
min.id = apply(!is.na(anmax.out), 1, sum) > 19
anmax.out2 = anmax.out[min.id,]
df.meta.out = df.meta[min.id,]

# save it. do NOT save the annual maxima to the data folder
write.csv(anmax.out2, paste0(idir,“atsite-anmax-“,dur,”-day.csv”), row.names=FALSE)
write.csv(df.meta.out, paste0(idir,“atsite-anmax-“,dur,”-day-metadata.csv”), row.names=FALSE)
